Agent(dict) → {Object}

Defines a representation of an agent (typically, a speaker).
Parameters:
Name Type Description
dict Array An associative array with keys credence, options, utility, alpha. credence is either a Distribution or a dictionary of states and their probabilities. options is a list of possible acts/utterances. utility is a function that takes an act and a state and returns a number. alpha is a number that, if set, controls the agent's softmax rationality; if unset, the agent maximizes expected utility.
Source:
  • rsa.wppl.js, line 46
See:
  • choice
Returns:
An agent with the specified properties;
Type
Object
Example
var speaker = Agent({
   credence: { 'rain': 0.2, 'sun': 0.8 },
`  options: ['A', 'B', 'C'],
   utility: function(act, state) { return act == 'A' ? (state == 'rain' ? 1 : 0) : 0; }
});
